О проекте

Что такое StudyBot?

StudyBot — индивидуальный учебный проект в рамках проектной практики первого курса. Цель — изучить технологию Telegram-ботов с нуля и реализовать полноценный рабочий инструмент для студентов.

«Лучший способ изучить технологию — построить с её помощью что-то полезное для самого себя.» — Принцип проекта
Проблема

Зачем это нужно

😩

Проблема

Студенты теряют концентрацию, забывают делать перерывы и не имеют удобного инструмента для быстрых вычислений и заметок прямо в мессенджере.

🚀

Решение

Telegram-бот, который всегда под рукой: мотивирует, ведёт таймеры, хранит заметки и помогает с конвертациями. Один интерфейс — много функций.

Технологии

Стек проекта

🐍

Python 3

Основной язык разработки. Простой синтаксис, огромная экосистема библиотек.

📦

pyTelegramBotAPI

Библиотека для работы с Telegram Bot API. Декораторный синтаксис, инлайн-кнопки.

🔐

python-dotenv

Хранение секретных токенов в файле .env вне репозитория.

💾

JSON

Постоянное хранилище заметок, расписания и статистики пользователей.

🌐

HTML & CSS

Сайт документации создан на чистом HTML и CSS, без фреймворков и сборщиков.

🗂

Git & GitHub

Управление версиями. Все изменения фиксируются осмысленными коммитами.

Цели

Что было изучено

  1. Работа с API

    Принципы REST API, формат JSON, методы Telegram Bot API.

  2. Python-разработка

    Декораторы, многопоточность, работа с файлами, обработка исключений.

  3. Безопасность

    Хранение секретов через переменные окружения и .gitignore.

  4. Работа с Git

    Создание репозитория, коммиты, ветки, пуш на GitHub.

  5. HTML/CSS вёрстка

    Многостраничный сайт без фреймворков, адаптивный дизайн, типографика.

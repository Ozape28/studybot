# StudyBot 🤖

Telegram-бот — учебный помощник для студентов. Реализован в рамках проектной (учебной) практики в Московском политехническом университете, направление ИСИТ.

## Возможности

- ⏱ **Таймер Помодоро** — живой счётчик с возможностью отмены
- 💪 **Мотивация** — случайные мотивирующие цитаты
- 🔢 **Конвертер единиц** — длина, масса, время, температура
- 📝 **Заметки** — сохранение и удаление через inline-кнопки
- 📅 **Расписание на день** — формат «9:00 Физика»
- 🎲 **Помощник выбора** — случайный выбор из вариантов
- 📊 **Статистика** — учёт завершённых таймеров
- 💾 **Постоянное хранилище** — данные сохраняются в JSON

## Установка и запуск

1. Установи Python 3.10+ с [python.org](https://python.org)
2. Клонируй репозиторий:
git clone https://github.com/ТВОЙ_ЛОГИН/studybot.git cd studybot

3. Установи зависимости:
pip install -r requirements.txt

4. Получи токен у [@BotFather](https://t.me/BotFather) в Telegram
5. Создай файл `.env` (см. `.env.example`) и впиши свой токен:
BOT_TOKEN=твой_токен_здесь

6. Запусти бота:
python bot.py


## Стек технологий

- **Python 3** + `pyTelegramBotAPI` — серверная часть
- **python-dotenv** — управление секретами
- **JSON** — хранилище данных пользователей
- **HTML + CSS** — сайт-документация (без фреймворков)

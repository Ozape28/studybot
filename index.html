Что умеет StudyBot

Восемь функций, которые делают учёбу чуть легче и приятнее каждый день.

⏱

Таймер Помодоро

Живой счётчик в Telegram с прогресс-баром, кнопкой отмены и автоматическим переходом на отдых.

💪

Мотивация

Случайные мотивирующие цитаты по запросу. Для тех, кто застрял и нуждается в пинке.

🔢

Конвертер единиц

Длина, масса, время, температура. Поддержка русских и английских названий.

📝

Заметки

Быстрое сохранение мыслей, удаление через инлайн-кнопки, всё хранится между сессиями.

📅

Расписание

Простое добавление дел в формате «9:00 Физика». Сортировка по времени автоматически.

🎲

Помощник выбора

Не можешь решить, что поесть? Закинь варианты — бот выберет случайно за тебя.

📊

Статистика

Сколько помидоров завершил, сколько отменил, сколько заметок и дел в расписании.

💾

Постоянное хранилище

Все данные пользователя сохраняются в JSON и не теряются после перезапуска бота.

Как запустить

Быстрый старт

  1. Установи Python 3

    Скачай с python.org. При установке отметь галочку «Add to PATH».

  2. Установи зависимости

    В командной строке: pip install pyTelegramBotAPI python-dotenv

  3. Получи токен у @BotFather

    Напиши /newbot в Telegram — получишь уникальный API-токен.

  4. Создай файл .env

    Положи туда BOT_TOKEN=твой_токен. Так токен не попадёт на GitHub.

  5. Запусти бот

    Выполни в терминале python bot.py — и бот в эфире.

$ pip install pyTelegramBotAPI python-dotenv
$ python bot.py
✅ Бот запущен. Ctrl+C для остановки.

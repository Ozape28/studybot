Все этапы проекта — от старта до финала

Сводная хронология задач, выполненных в ходе практики. Весь проект реализован индивидуально.

  • Изучение технологии Telegram Bot API (~6 ч)
  • Регистрация бота через @BotFather (~30 мин)
  • Разработка базовой логики на Python: /start, /help, /motivation, /pomodoro, /convert (~8 ч)
  • Расширение функционала: живой таймер с отменой, заметки, расписание, статистика, JSON-хранилище (~10 ч)
  • Защита токена через .env и .gitignore (~1 ч)
  • Дизайн и вёрстка пятистраничного сайта на HTML/CSS (~8 ч)
  • Написание документации в Markdown (~4 ч)
  • Настройка Git-репозитория и публикация на GitHub (~2 ч)
  • Подготовка отчёта по практике (~4 ч)

Сайт-документация готов

Завершил вёрстку многостраничного сайта на чистом HTML и CSS. Сайт включает пять страниц: главную с описанием функций, страницу о проекте с архитектурой, страницу участников, журнал прогресса и список ресурсов. В дизайне использованы шрифты Instrument Serif для заголовков и JetBrains Mono для технических элементов. Тёмная тема, плавные анимации, адаптивная вёрстка под мобильные устройства.

  • 5 HTML-страниц, единый style.css
  • Анимации появления и hover-эффекты
  • Адаптивный дизайн для экранов до 600px

Защита токена через .env

Узнал, что хранить токен прямо в коде — серьёзная уязвимость. Установил библиотеку python-dotenv, вынес BOT_TOKEN в отдельный файл .env, который добавил в .gitignore. Старый токен пришлось перевыпустить через @BotFather, так как он успел засветиться. Теперь в репозитории лежит только .env.example как шаблон без реальных секретов.

→ Запись #03 Расширение функционала

Большое обновление — Помодоро 2.0 и новые команды

Базовый бот оказался слишком простым: у таймера не было счётчика, нельзя было отменить его после запуска. Полностью переписал Помодоро — теперь сообщение редактируется каждые 30 секунд, показывает оставшееся время и прогресс-бар, а под ним инлайн-кнопка «Отменить». Добавил функции:

  • Заметки с возможностью удаления через кнопки
  • Расписание на день в формате «9:00 Физика»
  • Расширенный конвертер: длина, масса, время, температура
  • Команду /choose для случайного выбора из вариантов
  • Статистику завершённых помидоров
  • Постоянное хранилище данных в JSON

Базовый бот работает

Установил Python и библиотеку pyTelegramBotAPI. Зарегистрировал бота через @BotFather, получил токен. Написал первую версию bot.py с четырьмя командами: /start, /help, /motivation (случайная цитата) и /pomodoro (таймер на 25 минут через time.sleep в отдельном потоке). Также добавил конвертер единиц для километров, килограммов и часов. Бот успешно отвечает на сообщения в Telegram.

Выбор темы и подготовка

Изучил задание проектной практики. По варианту 2 нужно реализовать одну из технологий из репозитория build-your-own-x. Из вариантов остановился на Telegram-боте — это интересно, полезно и подходит для индивидуальной работы. Прочитал документацию Telegram Bot API, посмотрел несколько туториалов, сформировал идею: помощник для студентов с таймером, конвертером и мотивацией. Создал репозиторий на GitHub из шаблона курса.
